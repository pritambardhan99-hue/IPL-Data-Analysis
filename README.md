# 🏏 IPL Data Analysis Dashboard (2008–2025)

An interactive data analysis dashboard covering all **18 seasons** of the Indian Premier League (IPL) — from the inaugural 2008 season to the historic 2025 season where RCB won their first ever title.



## 📊 What's Inside

The dashboard has **6 interactive tabs**:

| Tab | Content |
|-----|---------|
| 📊 **Overview** | Season-wise matches, title winners, toss analysis, win types |
| 🏆 **Teams** | All-time wins, win percentage radar, full team table |
| 🏏 **Batsmen** | Top 10 run scorers, centuries vs fifties, full stats |
| 🎯 **Bowlers** | Top 10 wicket takers, economy rates, full stats |
| 📅 **All Seasons** | Complete 18-season summary with Orange & Purple Cap winners |
| 💡 **Insights** | Key facts, score distribution, venue stats, player nationalities |

## 📈 Key Findings

- **RCB finally won** their maiden IPL title in 2025 after 18 years and 3 final losses (2009, 2011, 2016)
- **MI & CSK** are joint most successful with **5 titles each**
- **KKR** won their **3rd title** in IPL 2024
- **Virat Kohli** holds the all-time run record with **8,661+ runs** — only player to cross 8000
- **Yuzvendra Chahal** leads all-time wickets with **205+ wickets**
- **SRH scored 287/3** vs RCB in IPL 2024 — highest ever team score in IPL
- **Chasing teams win 53.2%** of matches — batting second is slightly favored
- **Average scores have risen** every season — from 152 in 2008 to 172 in 2024
- **Chris Gayle** holds records for most sixes (357) and highest individual score (175* off 66 balls)

## 🚀 How to Run

### Option 1 — Direct Open
```
Just double-click IPL_Analysis_Dashboard.html in your file explorer
```

### Option 2 — Live Server (VS Code)
```
1. Install Live Server extension in VS Code
2. Right-click the HTML file
3. Click "Open with Live Server"
```
